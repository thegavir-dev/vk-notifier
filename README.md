# VK Notifier

## Описание
VK Notifier — WordPress-плагин для дублирования email-уведомлений сайта в сообщения ВКонтакте.

## Основные возможности
- Перехват email-уведомлений WordPress.
- Отправка уведомлений в личные сообщения или беседы VK.
- Режим пересылки всех писем или только писем из белого списка email.
- Настройка получателей VK, префикса сообщения и уведомления о вложениях.
- Журналирование отправок и ошибок VK API.
- Автоматическая ротация логов через WP-Cron.
- Файл `readme.txt` и структура плагина подготовлены для проверки WordPress.org.

## Установка
1. Скопировать папку `vk-notifier` в `wp-content/plugins/`.
2. Активировать плагин в админке WordPress.
3. Открыть `Настройки -> VK Notifier`.

## Настройки
- Токен VK.
- ID сообщества VK.
- Получатели VK.
- Пересылать: все письма или только письма из белого списка.
- Белый список email: используется только в режиме пересылки `Только письма из белого списка`.
- Префикс сообщения.
- Текст о вложениях.
- Уровень логирования.
- Хранить логи (дней): по умолчанию `30`; значение `0` отключает автоматическую ротацию.
- Режим отправки уведомлений.

## Использование
- Указать токен VK и получателей.
- Выбрать режим пересылки: все письма сайта или только письма на email из белого списка.
- Выполнить тестовую отправку на странице настроек.
- Проверять журнал на странице `VK Notifier Logs`.

## Хуки / Интеграции
- Интеграция с `wp_mail`.
- Интеграция с VK API `messages.send`.
- Поддержка WP-Cron для фоновой отправки уведомлений.
- WP-Cron событие `vk_notifier_cleanup_logs` для ежедневной очистки старых логов.
- WP-Cron событие `vk_notifier_async_send` для фоновой отправки уведомлений в режиме `Через WP-Cron`.

## Ограничения / Примечания
- Версия плагина: `1.0.0`.
- Лицензия: `GPL v2 or later`.
- Минимальная версия WordPress для `readme.txt`: `5.0`; протестировано до `6.9`.
- Для токена сообщества нужен корректный положительный `group_id`.
- Для бесед нужно использовать положительный `peer_id`.
- Отрицательные ID получателей не поддерживаются.
- При режиме пересылки `Все письма` белый список email игнорируется.
- При деинсталляции плагин удаляет таблицу логов и свои опции.
