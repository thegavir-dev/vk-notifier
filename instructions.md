# Инструкция по настройке плагина VK Notifier

Плагин пересылает уведомления вашего сайта (например, о новых заказах или заявках) в виде сообщений ВКонтакте — в личку или групповую беседу.

---

## Что понадобится

- Аккаунт ВКонтакте
- Сообщество ВКонтакте (группа или публичная страница) — именно оно будет отправлять сообщения
- Доступ к панели администратора WordPress

---

## Шаг 1. Создайте сообщество ВКонтакте

> Если сообщество уже есть — переходите к шагу 2.

1. Войдите в ВКонтакте.
2. В левом меню нажмите **«Сообщества»**, затем кнопку **«Создать сообщество»**.
3. Выберите тип — подойдёт **«Группа»**.
4. Придумайте название (например, «Уведомления сайта») и нажмите **«Создать»**.

---

## Шаг 2. Включите сообщения в сообществе

Это обязательный шаг — без него плагин не сможет отправлять уведомления.

1. Откройте страницу вашего сообщества.
2. Нажмите кнопку **«Управление»** (под обложкой, справа).
3. В меню слева выберите **«Сообщения»**.
4. Переключите **«Сообщения сообщества»** в положение **«Включены»**.
5. Нажмите **«Сохранить»**.

---

## Шаг 3. Получите токен сообщества

Токен — это «ключ», который разрешает плагину отправлять сообщения от имени сообщества.

1. В управлении сообществом выберите в меню слева **«Работа с API»**.
2. Перейдите на вкладку **«Ключи доступа»**.
3. Нажмите **«Создать ключ»**.
4. В появившемся окне обязательно поставьте галочку напротив пункта **«Сообщения»** (`messages`). Остальное можно не трогать.
5. Нажмите **«Создать»**.
6. Подтвердите действие через телефон или код, если ВКонтакте попросит.
7. Скопируйте появившийся ключ — это длинная строка из букв и цифр. Сохраните её, она понадобится на следующем шаге.

> ⚠️ Никому не передавайте этот ключ — с его помощью можно отправлять сообщения от имени вашего сообщества.

---

## Шаг 4. Узнайте ID вашего сообщества

1. Откройте страницу сообщества в браузере.
2. Посмотрите на адрес — например: `vk.com/club123456789`
3. Цифры в конце и есть ID сообщества. В данном примере — `123456789`.

Если вместо цифр красивое слово (например, `vk.com/my_shop`):
1. Откройте любой пост сообщества.
2. В адресной строке будет что-то вроде: `vk.com/my_shop?w=wall-123456789_1`
3. Цифры после `wall-` и до знака `_` — это ID.

> ❗ Записывайте ID **без минуса**. Только цифры: `123456789`, а не `-123456789`.

---

## Шаг 5. Введите данные в плагин

1. Войдите в панель администратора WordPress.
2. В меню слева найдите **VK Notifier** → **«Настройки»**.
3. Заполните поля:

| Поле | Что вписать |
|---|---|
| **Токен VK** | Ключ из шага 3 |
| **ID сообщества VK** | Цифры из шага 4 (без минуса) |

4. Нажмите **«Сохранить изменения»**.

---

## Шаг 6. Укажите, кому отправлять уведомления

Плагин может отправлять сообщения в личку конкретному пользователю или в групповую беседу.

### Вариант А — в личку пользователю

1. Откройте профиль нужного человека ВКонтакте.
2. Посмотрите на адрес в браузере:
   - Если там `vk.com/id123456789` — цифры и есть ID. Вставьте их в поле **«Получатели VK»**.
   - Если там красивое имя, например `vk.com/ivan_petrov` — перейдите на сайт [vk.com/foaf.php?id=](https://vk.com/foaf.php?screen_name=ivan_petrov), заменив `ivan_petrov` на нужное имя. Откроется страница с кодом — найдите строчку вида `<ya:userID>123456789</ya:userID>`. Цифры внутри тегов — это и есть ID.
   - Либо воспользуйтесь любым бесплатным онлайн-сервисом поиска ID ВКонтакте — достаточно ввести в поиске «узнать ID ВКонтакте по ссылке».

> ⚠️ Пользователь должен сам написать сообщество хотя бы один раз — иначе ВКонтакте не даст отправить ему сообщение (ошибка 901).

### Вариант Б — в групповую беседу

1. Перейдите в **VK Notifier** → **«Настройки»**.
2. Убедитесь, что токен и ID сообщества уже сохранены.
3. Нажмите кнопку **«Найти доступные беседы сообщества»**.
4. Плагин покажет список бесед, к которым у сообщества есть доступ.
5. Скопируйте значение `peer_id` нужной беседы.
6. Вставьте его в поле **«Получатели VK»**.

> Если беседа не появляется в списке — убедитесь, что сообщество добавлено в эту беседу как участник.

### Несколько получателей

Можно указать сразу нескольких — через запятую:
```
123456789, 2000000001
```

---

## Шаг 7. Настройте, какие письма пересылать

В поле **«Пересылать»** выберите один из режимов:

- **Только письма из белого списка** — пересылаются только письма, отправленные на адреса из поля «Белый список email». Укажите нужные адреса через запятую.
- **Все письма** — пересылаются все уведомления сайта без исключения. Поле белого списка при этом скрывается.

---

## Шаг 8. Проверьте, что всё работает

1. На странице настроек нажмите кнопку **«Отправить тестовое сообщение»**.
2. Если всё настроено верно — в указанный чат или личку придёт сообщение: _«Тестовое сообщение от VK Notifier. Если вы это читаете, плагин работает корректно.»_
3. Если сообщение не пришло — откройте **VK Notifier** → **«Логи»** и посмотрите на последние записи.

---

## Что делать, если что-то пошло не так

Откройте **VK Notifier** → **«Логи»**. Найдите последнюю запись с красной меткой «Ошибка» и посмотрите код ВКонтакте:

| Код ошибки | Причина | Решение |
|---|---|---|
| **901** | Пользователь не разрешил сообщения от сообщества | Попросите его написать сообществу первым |
| **917** | Сообщество не имеет доступа к беседе или неверный peer_id | Используйте кнопку «Найти доступные беседы» и берите peer_id оттуда |
| **932** | Сообщество не может писать этому получателю | Проверьте правильность ID получателя |

Если ошибки нет, но сообщения не приходят — убедитесь, что токен скопирован полностью, без лишних пробелов в начале или конце.

---

## Дополнительные настройки (по желанию)

| Настройка | Что делает |
|---|---|
| **Префикс сообщения** | Текст, который будет добавлен в начало каждого уведомления |
| **Текст о вложениях** | Добавляется, если в письме были файлы |
| **Режим отправки** | «Сразу» — надёжнее для большинства сайтов; «Через WP-Cron» — фоновая отправка, может задерживаться |
| **Уровень логирования** | «Все события» — записывать всё; «Только ошибки» — записывать только проблемы |
| **Хранить логи (дней)** | Сколько дней хранить историю отправок. `0` — хранить бессрочно |
